# chitose-iconfont
## How to use
### 1,Embed this
```
<link rel="stylesheet" href="https://chitose1020.github.io/chitose-iconfont/icons/icon.css">
```
### 2,Use!
You can create Element div
Please assign a class that corresponds to the icon you want to use.
